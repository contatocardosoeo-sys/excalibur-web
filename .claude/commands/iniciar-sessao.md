# Comando: /iniciar-sessao
# Cole no início de cada sessão do Claude Code

Execute automaticamente:

1. Leia CLAUDE.md completo
2. Leia AGENTS.md completo  
3. Leia PROMPTS_MESTRES.md completo
4. Leia STATUS.md — entenda onde o projeto está
5. Leia PROGRESSO.md — veja o que foi feito
6. Rode npm run build — verifique se está tudo funcionando
7. Abra localhost:3000 e tire screenshot do estado atual
8. Reporte ao CEO:
   - ✅ O que está funcionando
   - 🔄 O que estava em andamento
   - ➡️ O que fazer agora
9. Aguarde ordem do CEO para iniciar próxima task
